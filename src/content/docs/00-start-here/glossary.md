# Glossary

Glossary adalah daftar istilah. Tujuannya membantu pembaca memahami kata teknis yang sering muncul dalam dokumentasi.

## Istilah Umum

| Istilah | Arti Sederhana |
|---|---|
| Program | Kumpulan perintah yang dijalankan komputer atau mikrokontroler. |
| Source code | Teks berisi instruksi program yang ditulis manusia. |
| File | Satu berkas data, misalnya `main.cpp` atau `platformio.ini`. |
| Folder | Tempat untuk mengelompokkan file. |
| Project | Kumpulan file dan folder untuk membangun satu sistem. |
| Build | Proses mengubah source code menjadi program yang bisa dijalankan. |
| Debugging | Proses mencari penyebab masalah. |
| Log | Catatan kejadian yang ditulis program untuk membantu debugging. |

## Istilah IoT dan Firmware

| Istilah | Arti Sederhana |
|---|---|
| IoT | Internet of Things, yaitu perangkat fisik yang terhubung ke jaringan. |
| WSN | Wireless Sensor Network, jaringan sensor nirkabel. |
| Firmware | Program yang berjalan di perangkat seperti ESP8266 atau ESP32. |
| Mikrokontroler | Chip kecil yang bisa membaca sensor dan mengendalikan perangkat. |
| Node sensor | Perangkat yang membaca kondisi lingkungan greenhouse. |
| Gateway | Perangkat penghubung dan pengendali lokal. |
| Sensor | Alat untuk membaca kondisi fisik, seperti suhu atau cahaya. |
| Aktuator | Alat yang melakukan aksi fisik, seperti relay atau fan. |
| Relay | Saklar elektronik yang dikendalikan program. |
| OTA | Over The Air update, pembaruan firmware lewat jaringan. |
| Captive portal | Halaman konfigurasi yang muncul saat perangkat membuat Wi-Fi lokal. |

## Istilah Backend, Web, dan Android

| Istilah | Arti Sederhana |
|---|---|
| Backend | Program server yang menerima, mengolah, dan menyimpan data. |
| Laravel | Framework PHP untuk membuat backend. |
| Frontend | Tampilan web yang dilihat pengguna. |
| Vue.js | Framework JavaScript untuk membuat tampilan web interaktif. |
| Android WebView | Komponen Android yang membuka halaman web di dalam aplikasi. |
| API | Jalur komunikasi antar program. |
| Request | Permintaan dari client ke server. |
| Response | Jawaban dari server ke client. |
| Endpoint | Alamat API tertentu, misalnya untuk mengirim data sensor. |

## Istilah Database

| Istilah | Arti Sederhana |
|---|---|
| Database | Tempat menyimpan data secara terstruktur. |
| MySQL | Software database relasional. |
| Tabel | Kelompok data dengan kolom dan baris. |
| Kolom | Jenis data dalam tabel, seperti `temperature` atau `created_at`. |
| Baris | Satu data lengkap dalam tabel. |
| Primary key | Identitas unik setiap baris. |
| Foreign key | Kolom yang menghubungkan satu tabel dengan tabel lain. |
| Query | Perintah untuk membaca atau mengubah database. |

## Istilah Jaringan dan Keamanan

| Istilah | Arti Sederhana |
|---|---|
| Wi-Fi | Jaringan nirkabel untuk menghubungkan perangkat. |
| HTTPS | HTTP yang memakai perlindungan enkripsi. |
| WebSocket | Jalur komunikasi dua arah yang bisa tetap terbuka. |
| Token | Kode rahasia untuk membuktikan identitas atau izin akses. |
| AES-256-CBC | Cara enkripsi data dengan kunci 256-bit dan mode CBC. |
| Enkripsi | Mengubah data agar tidak mudah dibaca pihak lain. |
| Autentikasi | Proses membuktikan siapa pengguna atau perangkat. |
| QoS | Quality of Service, ukuran kualitas jaringan seperti delay dan packet loss. |

## Catatan

Beberapa istilah akan dijelaskan lebih panjang di halaman khusus. Glossary ini hanya memberi arti cepat agar pembaca tidak tersesat saat membaca halaman lain.
