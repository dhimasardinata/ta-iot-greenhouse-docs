# Manfaat

Manfaat menjelaskan siapa yang terbantu oleh sistem dan dokumentasi.

## Manfaat untuk Pengelola Greenhouse

Sistem dapat membantu pengelola melihat kondisi greenhouse melalui data. Data seperti suhu, kelembapan, dan cahaya dapat dipantau melalui dashboard.

Jika sistem kontrol aktuator berjalan sesuai desain, pengelola juga dapat terbantu dalam menjaga kondisi mikro iklim greenhouse.

## Manfaat untuk Penelitian TA

Sistem memberi bahan untuk menjelaskan:

- rancangan IoT-WSN,
- alur node ke cloud,
- alur node ke gateway,
- alur gateway ke aktuator,
- API dan database,
- dashboard web dan Android,
- pengujian QoS, data loss, delay, throughput, dan RSSI jika data uji tersedia,
- keamanan komunikasi.

## Manfaat untuk Developer Baru

Dokumentasi file-by-file membantu developer baru mengetahui:

1. file mana yang penting,
2. file mana yang boleh diabaikan,
3. data apa yang masuk dan keluar,
4. efek samping setiap file,
5. cara debugging,
6. risiko saat mengubah kode.

## Manfaat untuk Pembaca Awam

Pembaca awam dapat belajar bertahap. Dokumentasi tidak langsung memaksa pembaca memahami C++, PHP, Vue, Kotlin, database, dan jaringan sekaligus.

## Manfaat untuk Maintenance

Sistem IoT biasanya terus berubah. Ada sensor yang perlu dikalibrasi, endpoint yang berubah, firmware yang diperbarui, atau perangkat yang dipindah. Dokumentasi membantu perubahan dilakukan lebih aman.

Lanjutkan ke [Metodologi Prototyping](./metodologi-prototyping.md).
