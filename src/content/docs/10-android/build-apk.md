# Build APK

Build APK adalah proses membuat file aplikasi Android yang bisa dipasang.

## Status Bukti

Snapshot awal hanya berisi tiga file Android text. File Gradle, project structure, resource lengkap, dan Firebase config tidak terlihat.

## Yang Belum Terkonfirmasi

- `build.gradle`,
- Android Gradle Plugin version,
- minSdk/targetSdk,
- package name final,
- Firebase config,
- icon resource,
- theme resource,
- service `MyFirebaseMessagingService`.

## Catatan

Dokumentasi build APK lengkap baru bisa dibuat setelah project Android lengkap tersedia.

Lanjutkan ke [Debugging Android](./debugging-android.md).
