# node/.codacy.yml

File ini mengatur analisis kualitas kode di Codacy untuk proyek firmware node.

## Metadata File

| Item | Nilai |
|---|---|
| Source file | `node/.codacy.yml` |
| Komponen | Config |
| Level | Menengah |
| Status | Drafted |
| Terakhir diperiksa | 2026-05-19 |

## Kenapa File Ini Ada

Codacy membaca file ini untuk tahu alat analisis mana yang aktif dan folder mana yang tidak perlu diperiksa. Tujuannya adalah menangkap masalah C/C++ tanpa membuang waktu pada dependency, hasil build, dan dokumentasi.

## Engine yang Aktif

| Engine | Status | Fungsi |
|---|---|---|
| `cppcheck` | Aktif | Mencari masalah umum pada kode C/C++. |
| `clang-tidy` | Aktif | Mencari pola kode yang berisiko atau tidak sesuai praktik baik. |

## Path yang Dikecualikan

File ini mengecualikan:

- `test/**`
- `lib/**/test/**`
- `.pio/**`
- `build/**`
- `node_modules/**`
- `docs/**`

Jadi analisis Codacy difokuskan ke source utama, bukan test, build output, dependency JavaScript, atau dokumentasi.

## Bahasa yang Diperiksa

Bagian `languages.cpp.extensions` mencakup:

- `.cpp`
- `.h`
- `.hpp`
- `.ino`

Ini sesuai dengan bentuk file firmware Arduino/PlatformIO.

## Error yang Mungkin Terjadi

- Jika source penting masuk folder yang dikecualikan, Codacy tidak akan memeriksanya.
- Jika dependency atau output build tidak dikecualikan, hasil analisis bisa ramai dengan masalah yang bukan milik proyek.
- Jika engine Codacy berubah versi, temuan baru bisa muncul walau source tidak berubah.

## Bagian untuk Pemula

File ini seperti daftar pemeriksaan otomatis. Codacy memakai daftar ini untuk mencari bagian kode yang mungkin bermasalah sebelum firmware dipakai lebih jauh.

## Bagian Advanced

Pengecualian `test/**` membuat hasil Codacy lebih fokus ke firmware utama, tetapi risiko di test helper tidak akan terlihat dari konfigurasi ini. Untuk audit test, perlu pengecekan terpisah.

## Hubungan ke Sistem TA

Firmware node mengirim data sensor dan menerima perintah. Analisis statis membantu mengurangi risiko bug sebelum firmware dijalankan di perangkat greenhouse.
