# Folder Overview: Android

Folder ini akan menampung dokumentasi file-by-file untuk aplikasi Android WebView.

## Fungsi Komponen

Android WebView membuka sistem web melalui aplikasi mobile. Komponen ini penting agar pengguna bisa mengakses dashboard dari ponsel.

## File yang Terlihat di Inventory

- `android/AndroidManifest.xml.txt`
- `android/MainActivity.kt.txt`
- `android/activity_main.xml.txt`

## Hal yang Harus Dicari Saat File-by-File

- permission aplikasi,
- Activity utama,
- URL yang dimuat WebView,
- handling loading,
- handling error,
- layout XML,
- perilaku tombol back jika ada.

## Urutan Membaca

1. `android/AndroidManifest.xml.txt`
2. `android/activity_main.xml.txt`
3. `android/MainActivity.kt.txt`

## Status

Tiga file Android sudah memiliki halaman detail awal dengan status `Drafted` di [Coverage Report](../coverage-report.md). Review berikutnya perlu membandingkan halaman ini dengan source Android asli jika file `.txt` diganti menjadi proyek Android lengkap.
